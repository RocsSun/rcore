# RCore OS 开发的环境搭建

主要环境是rust开发，因此以rust为主。

当前环境：

- Debian 13.4


## 安装基础环境

Debian 13的qemu版本10.0，不需要重新编译安装

```sh
$ sudo apt update && sudo apt upgrade -y && sudo apt autoremove -y
$ sudo apt install git build-essential gdb-multiarch qemu-system-misc gcc-riscv64-linux-gnu binutils-riscv64-linux-gnu qemu-user opensbi u-boot-qemu -y
```

## 安装RISCV GDB

```shell
$ sudo apt install libgmp-dev libmpfr-dev -y
$ 
$ curl -O https://mirrors.tuna.tsinghua.edu.cn/gnu/gdb/gdb-17.1.tar.xz
$ tar -xf  gdb-17.1.tar.xz && cd gdb-17.1/
$ mkdir build && cd build

$ ../configure --prefix=/usr/local --target=riscv64-unknown-elf --enable-tui=yes

$ make -j$(nproc)
$ sudo make install
```

参考资料：
 
[MIT6.S081 riscv64-unknown-elf-gdb安装指北](https://zhuanlan.zhihu.com/p/638731320)

## RUST安装

rust安装参考[RsProxy.cn](https://rsproxy.cn/#getStarted)

其余的rust包，通过项目下的`rust-toolchain.toml`文件控制，内容如下：

```toml
[toolchain]
channel = "nightly-2026-01-10"
targets = ["riscv64gc-unknown-none-elf"]
components = ["cargo-binutils", "llvm-tools-preview", "rust-src"]
```
